Configurar o Git

Isso é importante para sistemas de controle de versão, pois cada commit do Git usa essas informações:

Nome de usuário

Seu nome será anexado aos seus commits. Defina-o com:

  • git config --global user.name "Sombra da noite"

Endereço de email

Seu e-mail também é anexado aos seus commits. Defina-o com:

  • git config --global user.email "sombra123@exemplo.com"

Você provavelmente também vai querer usar isso ao se registrar no GitHub mais tarde.

  • Use --global para definir o valor para cada repositório no seu computador.
  • Use --local (o padrão) para defini-lo somente para o repositório atual.

Visualizando sua configuração

  • git config --list

Exibir uma configuração específica

  • git config user.name

Alterar ou desconfigurar valores de configuração

  • Para alterar um valor, basta executar o git config comando novamente com o novo valor.
  • Para remover uma configuração, use --unset:

Desfazer a definição de um alias

  • git config --global --unset code.editor

Nome da filial padrão

Defina o nome do branch padrão para novos repositórios (por exemplo, main em vez de master):

  • git config --global init.defaultBranch main

Níveis de configuração

Existem três níveis de configuração:

  • Sistema (todos os usuários):git config --system
  • Global (usuário atual):git config --global
  • Local (repositório atual):git config --local

A ordem de precedência é:

  • Local (repositório atual)
  • Global (usuário atual)
  • Sistema (todos os usuários)

O motivo para usar níveis diferentes é que você pode definir valores diferentes para usuários ou repositórios diferentes.

Isso pode ser usado, por exemplo, para definir diferentes ramificações padrões para diferentes repositórios e usuários.

definir uma configuração local

As configurações locais se aplicam somente ao repositório atual.

  • git config user.name "Project Name"

definir uma configuração global

As configurações globais se aplicam a todos os repositórios do usuário atual.

  • git config --global user.name "Global Name"

definir uma configuração do sistema

As configurações do sistema se aplicam a todos os repositórios para todos os usuários.

  • git config --system user.name "System Name"
